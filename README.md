# Smoke

Painel administrativo e vitrine pública da loja Smoke.

## Stack

- TanStack Start (React 19, Vite)
- Tailwind CSS v4
- Lovable Cloud (Supabase) — Auth, Postgres, RLS, Storage

## Desenvolvimento

```bash
bun install
bun run dev
```

## Deploy

Publique pelo Lovable. As tabelas operacionais iniciam vazias e são
populadas pelo próprio uso do painel.
